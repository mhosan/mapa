Controles geográficos

Este plugin envia el plano activo a la página de ARBA para que se le apliquen los controles geográficos y muestra los resultados dentro del programa cad.

Cálculo área polígonos en layer

Este plugin calcula el área de todas las polílineas cuyo punto de inicio coincida con el punto final (polígonos cerrados), que se encuentran en una determinada capa (layer).

La versión trial (de prueba) funciona por completo, sin restricciones, en forma gratuita, por siete dias.

Cambiar todos los textos a mayusculas o minusculas

Este plugin cambia todos los textos de un plano, o todos aquellos que se encuentren en una determinada capa (layer|), a mayusculas o minusculas

La versión trial (de prueba) funciona por completo, sin restricciones, en forma gratuita, por siete dias.

Este plugin está diseñado para los profesionales de la Agrimensura de la Provincia de Buenos Aires, Argentina, los cuales envian archivos cad en formato DXF a la Agencia de Recaudación de la Provincia (ARBA).

Este plugin es de uso genérico. Proporciona información de las áreas de los polígonos que se encuentran en el layer seleccionado y realiza la sumatoria de todas las áreas. Dispone de una opción para recalcular en caso de modificaciones tanto en la cantidad de poligonos como en el área de cada uno.

Para instalar complementos (plugins):

Cada complemento (plugin) es un archivo con extensión ".dll". Por ejemplo el que cálcula áreas es PoligonosAreas.dll.

Cuando se realiza la instalación aceptando las opciones por defecto, este archivo de tipo dll quedará en la carpeta "Archivos de programa/Argengis/Poligonos Areas/PoligonosAreas.dll" (en el caso de este ejemplo).

Es importante tener en claro la ubicación del complemento porque para utilizarlo hay que tipear en la línea de comandos del cad el comando "Netload" y presionar Enter.

Este comando muestra una pantalla de selección de archivos, propia de Windows, y en esa pantalla hay que localizar y seleccionar este archivo dll ("PoligonosAreas.dll" en este ejemplo). Con esto el complemento quedá incorporado al cad como un comando más.

Por último, en la línea de comandos del cad, escribir "calcarea" (para este ejemplo) y presionar Enter para comenzar la ejecución del complemento (o plugin).

Al finalizar la instalación de cualquier plugin, queda una carpeta en el escritorio de Windows con el nombre del plugin descargado, conteniendo un archivo de texto con instrucciones.
