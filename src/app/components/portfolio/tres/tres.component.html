  • Desarrollo personal: Visor gis Web de mapas.
    • Permite seleccionar, en la Pcia. de Bs. As, Partidos, escuelas y circuitos electorales.
    • Contiene un geolocalizador con dos opciones:
      • Local, con la Api de la Secretaria de Modernización dependiente de la Presidencia de la Nación - Argentina (https://datos.gob.ar/apis)
      • Global con la libreria LocationIQ: https://locationiq.com/ Incluye gráficos COVID-19 (Chartjs). Hecho con Angular 9, Leaflet 1.4.0 y Bootstrap 4.4
    • Hecho con Angular 9, Leaflet 1.4.0 y Bootstrap 4.4.
    • Persiste datos en Firebase. Es una aplicación web con backend como servicio (BaaS): Firebase proporciona servicios backend preconfigurados y escalables.
    • Se usó mapbox/leaflet-omnivore para que Leaflet admitiera los formatos CSV, GPX, KML, WKT y TopoJson.
    • Url: https://argengis.web.app
  • Desarrollo personal: Imagen Docker con visor gis web. Año 2018.
    • Hecho con Angular 6, Leaflet 1.4, Bootstrap y Ngnix:
    • Imagen en Docker Hub: hub.docker.com/mhosan
    • Para ejecutar, en local, ejecutar el comando “docker run -p 8087:80 mhosan/docker-angular-map” y luego, en el navegador, utilizar la url “localhost:8087”.
