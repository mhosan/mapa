Proyectos

  • ARBA (Agencia de Recaudación de la Provincia de Buenos Aires): Desarrollos varios. Entre ellos visores gis web propositos generales
  • Municip. La Plata: Sistema control certificaciones obras viales geolocalizadas.
  • C.P.A: Desarrollo aplicación CAD para el Consejo Prof. de Agrim de la Pcia. de Bs.As. 2008 - 2013
  • Desarrollo personal: Visor gis Web de mapas. Hecho con Angular 9, Leaflet 1.4.0 y Bootstrap 4.4. Persiste datos en Firebase. Es una aplicación web con backend como servicio (BaaS): Firebase proporciona servicios backend preconfigurados y escalables. Se usó mapbox/leaflet-omnivore para que Leaflet admitiera los formatos CSV, GPX, KML, WKT y TopoJson. https://argengis.web.app
  • Desarrollo personal:Imagen Docker con visor gis web hecho con Angular, Leaflet, Bootstrap y Ngnix: hub.docker.com/mhosan Para ejecutar, en local, ejecutar el comando “docker run -p 8087:80 mhosan/docker-angular-map” y luego, en el navegador, utilizar la url “localhost:8087”.
